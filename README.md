# 🍽️ Lyraa — Fine Dining Restaurant Website

Lyraa is a premium, dark-luxury fine dining restaurant website built with Next.js. It features a sophisticated design, smooth animations, and a seamless user experience across all devices.

## ✨ Features
✦ **Full screen luxury hero** with parallax and high-end typography  
✦ **Animated menu** with category filters (Starters, Mains, Desserts, Drinks)  
✦ **Live countdown timer** for exclusive seasonal offers  
✦ **Masonry photo gallery** with interactive lightbox and keyboard navigation  
✦ **Working reservation form** with complex validation (Mondays disabled, time slot selection)  
✦ **Elegant testimonials carousel** with mobile swipe support  
✦ **Responsive design** optimized for mobile, tablet, and desktop  
✦ **Premium dark luxury aesthetic** with candle-glow effects and gold accents  

## 🛠️ Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Forms**: React Hook Form
- **Icons**: React Icons
- **Language**: JavaScript

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/lyraa.git
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## 📦 Deployment
The easiest way to deploy Lyraa is to use the [Vercel Platform](https://vercel.com/new).

1. Push your code to a GitHub repository.
2. Import the repository into Vercel.
3. Vercel will automatically detect Next.js and deploy your site.

Alternatively, you can use the Vercel CLI:
```bash
npx vercel
```
